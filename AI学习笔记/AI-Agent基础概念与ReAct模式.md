# AI Agent 基础概念与 ReAct 模式

## Agent 是什么

AI Agent 是一个能够自主感知环境、做出决策、执行行动的智能体。核心公式：

> Agent = LLM + 工具 + 记忆 + 规划

## ReAct 模式

ReAct (Reasoning + Acting) 是目前最主流的单 Agent 循环模式：

1. Thought（思考）：分析当前状态，决定下一步
2. Action（执行）：调用工具或执行操作
3. Observation（观察）：获取执行结果
4. 循环直到任务完成

## 与传统程序的区别

传统程序：确定性的输入→处理→输出
AI Agent：非确定性的感知→推理→行动→观察→再推理

## 关键设计点

- 循环终止条件：最大迭代次数 + 任务完成判断
- 工具可插拔：统一 Tool 接口，运行时注册
- 异常处理：每步都可能失败，需要容错兜底

### 补充：循环终止条件\n- 最大迭代次数达到上限\n- LLM 输出中包含 FINAL_ANSWER 标记\n- 连续 3 轮工具调用返回相同结果（死循环检测）
