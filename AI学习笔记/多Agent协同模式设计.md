# 多 Agent 协同模式设计

## 7 种协同模式

| 模式 | 说明 | 适用场景 |
|------|------|------|
| 并行 | 多 Agent 同时处理不同子任务 | 独立任务批量处理 |
| 串行 | Agent A → Agent B → Agent C | 流水线处理 |
| 接力 | A 完成后交接给 B 继续 | 上下文传递 |
| 主从 | 主 Agent 分发，从 Agent 执行 | 任务委派 |
| 条件分支 | 根据条件路由到不同 Agent | 意图分发 |
| 循环迭代 | 结果不满足时重新处理 | 质量兜底 |
| 人工审批 | 关键节点需人工确认 | 安全敏感操作 |

## 任务拆解策略

### 规则模板（覆盖 80%）
- 预定义任务类型 → 固定拆解模板
- 低延迟、可预测

### LLM 实时拆解（20% 降级）
- 复杂/未知任务 → LLM 动态拆解
- DAG 无环校验 → 防止循环依赖

## 通信机制

Agent 间不直接通信，通过调度中枢（EventBus）中转：
- 解耦：Agent 不知道其他 Agent 的存在
- 可观测：所有消息可记录追踪
- 容错：单 Agent 故障不影响其他
