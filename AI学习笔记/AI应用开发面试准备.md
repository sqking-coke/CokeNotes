# AI 应用开发面试准备

## 高频考点

### Agent 核心
- Agent 循环原理（ReAct / Plan-Execute）
- Tool Calling 机制
- 多 Agent 协同模式
- 记忆管理方案

### RAG 系统
- 全链路：上传→分块→向量化→检索→生成
- 多路召回融合
- Rerank 精排
- 评估体系（Recall@K、忠实度）

### 框架理解
- LangChain4j vs 手写
- 框架的边界在哪里
- 什么时候用框架、什么时候不用

### 安全设计
- Prompt Injection 防护
- NL2SQL 安全防线
- API Key 管理
- 多租户隔离

### 工程化
- Docker 部署方案
- 可观测性（日志/指标/追踪）
- 模型可拔插架构
- 配置管理（环境变量/数据库/热更新）

## 面试叙事策略

1. 先讲手写（证明理解原理）
2. 再讲框架重写（证明工程效率）
3. 最后讲对比（证明判断力）
